API Description
https://hello-world-tiny-waterfall-cc7a.abinab.workers.dev This is used to report a player. For reporting you need to define type as report in body while sending the request. This is also used to update the status of the report data. For that send type as status.
https://blocklist.abinab.workers.dev This is block list api. If we make a post request with type= "block" then we can block a user as well as we can unblock by sending type="unblock" . For both we need to pass some extra details like the user Id, message as well as a Authentication Key.
https://blocklist.abinab.workers.dev?type=one&playerId=${playerId} This API is used to check if a player is in blocklist or not.
https://blocklist.abinab.workers.dev?type=all This API is used to get all the player data those are blocked.
DATABASE Description
Collection : Report , DB Name : Gamesapp The collection report contains all the reported players data. It contains the player who reported, reason, reportId, Status and Message.
Collection : blocklist , DB Name : Gamesapp The collection contains all the blocked user data.
Collection : unblock , DB Name : Gamesapp The collection contains all the data of those who are unblocked after blocking them.
